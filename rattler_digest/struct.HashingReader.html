Struct HashingReader 

Source 
pub struct HashingReader<R, D: Digest> { /* private fields */ }
Expand description

A simple object that provides a Read implementation that also immediately hashes the bytes read from it. Call HashingReader::finalize to retrieve both the original impl Read object as well as the hash.

If the tokio feature is enabled this object also implements [::tokio::io::AsyncRead] which allows you to use it in an async context as well.
