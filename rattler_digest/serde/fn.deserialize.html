Deserialize the [Output] of a [Digest].

Accepts either a hex-encoded string (for human-readable formats like JSON) or raw bytes (for binary formats like MessagePack). The format is detected per-value rather than by branching on deserializer.is_human_readable(), because serde’s internal Content deserializer (used for #[serde(flatten)] and friends) hardcodes is_human_readable() == true regardless of the underlying format, so a flattened digest field reached via a binary deserializer would otherwise be asked for a hex string.
