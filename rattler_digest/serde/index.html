Provides custom serialization/deserialization functions for [Output] of a [Digest].

Use the struct SerializableHash to easily serialize digest outputs. This type automatically handles both human-readable formats (like JSON) and binary formats (like MessagePack). See SerializableHash for detailed documentation on the serialization behavior.
